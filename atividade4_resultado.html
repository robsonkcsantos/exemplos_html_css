Crie um campo <input type="text"> e aplique os seguintes controles com base nos eventos abaixo:

  1. No evento de foco modifique o background do input para amarelo.
  2. Quando o campo perder o foco, recupere o seu respectivo valor e:
    • Caso o conteúdo contido no campo tenha menos de 3 caracteres o mesmo deve ter seu background alterado para vermelho.
    • Caso o conteúdo contido no campo tenha 3 caracteres ou mais o background deve ser alterado para verde.
